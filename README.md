# Syntecxhub_Website_Performmance_Analysis
## Project Overview
This project analyzes website traffic and user engagement using Python. By processing raw export data, I created an automated dashboard and several deep-dive visualizations to understand how different marketing channels drive traffic and which hours of the day yield the highest user engagement.

## Technologies Used

Language: Python

Libraries: Pandas (Data Manipulation), Matplotlib & Seaborn (Data Visualization)

Environment: Jupyter Notebook

## Key Objectives & Visualizations
In this notebook, I cleaned the raw dataset (handling datetime conversions and missing values) to generate the following insights:

Daily Traffic Trends: A time-series analysis comparing daily 'Users' vs. 'Sessions'.

Channel Performance: Bar charts identifying which Channel Groups drive the highest volume of traffic and the highest average engagement time.

Engagement Distribution: Boxplots showcasing the spread and consistency of engagement rates across different channels.

Engaged vs. Non-Engaged Traffic: Stacked/Melted bar charts to quickly visualize the quality of sessions per channel.

Traffic Heatmap: A 24-hour matrix revealing the peak hours for website sessions broken down by channel.
